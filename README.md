# whatslectron

**An electron app for Whatsapp web on Ubuntu Touch with responsive design. **  

[![OpenStore](https://open-store.io/badges/en_US.svg)](https://open-store.io/app/whatslectron.pparent)

## 🔨 Build Instructions

*need node.js 22.22.2*

### Standard

First, open/set a terminal in \Whatslectron-UT\whatslectron-src

Then run:
```bash
npm install
```
(optional) See if it launches:  
```bash
npm start
```
If it launches with no errors you can build it:
```bash
npm run build
```
The setup should be in dist\whatslectron Setup 1.0.0.exe
```bash
dist\whatslectron Setup 1.0.0.exe
```
(in dist\win-unpacked\ is the portable version)
